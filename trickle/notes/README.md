# Ligas Virtuales MX

## Overview
A comprehensive virtual soccer league management and simulation application.
Simulates 6 distinct leagues with matches, standings, and events.

## Features
- **Simulation Engine**: Automatic match scheduling and live minute-by-minute simulation.
- **League System**: Supports multiple leagues (1-5 standard, 6 grouped).
- **Admin Panel**: Secure area to manage teams and simulation flow.
- **Data Persistence**: Uses LocalStorage to save season progress.

## Tech Stack
- Frontend: React 18
- Styling: Tailwind CSS
- Icons: Lucide (Static)

## Setup
No build process required. Serve `index.html` via any static file server.
